website.

- **Spectral** — © The Spectral Project Authors (Production Type, for Google).
  Source: <https://github.com/productiontype/Spectral>

Only the `latin` and `latin-ext` subsets (woff2) are included, for the weights/styles the
site actually uses — Jost 300/400/500, Spectral 300/400 and 300 italic. The full OFL text
accompanies each font at the sources above. Generated from the Google Fonts CSS, then
served locally so no visitor IP is sent to a third-party CDN (GDPR).
